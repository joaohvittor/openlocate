# 🤝 Contributing to OpenLocate

Obrigado por considerar contribuir com o OpenLocate.

Este projeto tem um propósito **humanitário e sensível**, voltado ao apoio de investigações autorizadas envolvendo pessoas desaparecidas.

Por isso, seguimos regras claras de contribuição.

---

## ⚠️ Antes de começar

Este NÃO é:

- um sistema de vigilância
- uma ferramenta de identificação automática
- um mecanismo de busca facial em massa

Todas as contribuições devem respeitar:

✔ Privacidade  
✔ Uso ético  
✔ Conformidade legal  

---

## 🧠 Tipos de contribuição aceitos

- Melhorias no código (frontend/backend)
- Otimização de algoritmos de visão computacional
- Documentação
- Segurança e compliance
- UI/UX para ambientes investigativos
- Testes e validação

---

## 🚫 Contribuições NÃO permitidas

Não serão aceitas contribuições que:

- Implementem identificação automática
- Criem ou integrem bancos biométricos massivos

# 🤝 Contributing to OpenLocate

Thank you for your interest in contributing to OpenLocate.

This project has a **humanitarian and sensitive purpose**, focused on supporting legally authorized investigations involving missing persons, especially missing children.

For this reason, we follow strict and clear contribution guidelines.

---

## ⚠️ Before You Start

This project is NOT:

- a surveillance system  
- an automatic identification tool  
- a mass facial search engine  

All contributions must respect:

✔ Privacy  
✔ Ethical use  
✔ Legal compliance  

---

## 🧠 Accepted Contributions

We welcome contributions in areas such as:

- Frontend and backend improvements  
- Computer vision and machine learning optimization  
- Documentation  
- Security and compliance  
- UI/UX for investigative environments  
- Testing and validation  

---

## 🚫 Not Allowed

Contributions will be rejected if they:

- Implement automatic identification  
- Create or integrate large-scale biometric databases  
- Enable mass surveillance  
- Automate investigative decision-making  
- Bypass human validation  

---

## 🛠 How to Contribute

1. Fork the repository  
2. Create a new branch:

```bash
git checkout -b feature/your-feature-name
