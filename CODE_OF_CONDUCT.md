
# 📜 Code of Conduct — OpenLocate

## 🌍 Nosso compromisso

O OpenLocate é um projeto com foco humanitário.

Estamos comprometidos em criar um ambiente:

- seguro
- respeitoso
- ético
- colaborativo

---

## 🤝 Comportamento esperado

Todos os participantes devem:

- Tratar outros com respeito
- Aceitar críticas construtivas
- Manter discussões profissionais
- Priorizar o impacto social positivo
- Respeitar privacidade e sensibilidade do tema

---

## 🚫 Comportamento inaceitável

Não será tolerado:

- Assédio ou discriminação
- Uso do projeto para fins ilegais
- Tentativas de desvirtuar o propósito (ex: vigilância em massa)
- Compartilhamento de dados sensíveis sem autorização
- Linguagem ofensiva ou desrespeitosa

---

## ⚠️ Uso indevido

Qualquer tentativa de:

- transformar o projeto em sistema de vigilância
- criar identificação automática
- explorar vulnerabilidades

resultará em:

❌ bloqueio imediato  
❌ remoção de contribuições  

---

## 🛡 Aplicação

Os mantenedores têm o direito de:

- remover contribuições
- fechar issues
- bloquear usuários

quando necessário para proteger o projeto.

---

## 📩 Reportar problemas

Se você presenciar comportamento inadequado:

👉 Entre em contato com os mantenedores  
👉 Não exponha publicamente sem necessidade  

---

## 🌱 Cultura do projeto

Queremos construir:

✔ tecnologia responsável  
✔ colaboração saudável  
✔ impacto real  

---

## 💬 Nota final

Este não é apenas um projeto de código.

É um projeto que pode impactar vidas reais.

Aja de acordo com isso.

# 📜 Code of Conduct — OpenLocate

## 🌍 Our Commitment

OpenLocate is a project with a strong humanitarian focus.

We are committed to fostering an environment that is:

- safe  
- respectful  
- ethical  
- collaborative  

---

## 🤝 Expected Behavior

All participants are expected to:

- Treat others with respect  
- Accept constructive feedback  
- Maintain professional discussions  
- Prioritize positive social impact  
- Respect privacy and the sensitive nature of this domain  

---

## 🚫 Unacceptable Behavior

The following will not be tolerated:

- Harassment or discrimination  
- Use of the project for illegal purposes  
- Attempts to distort the project’s purpose (e.g., mass surveillance)  
- Sharing sensitive data without authorization  
- Offensive or disrespectful language  

---

## ⚠️ Misuse

Any attempt to:

- turn this project into a surveillance system  
- implement automatic identification  
- exploit vulnerabilities  

will result in:

❌ immediate restriction or ban  
❌ removal of contributions  

---

## 🛡 Enforcement

Maintainers have the right to:

- remove contributions  
- close issues or pull requests  
- restrict or block users  

when necessary to protect the integrity of the project.

---

## 📩 Reporting Issues

If you witness inappropriate behavior:

👉 Contact the maintainers directly  
👉 Avoid unnecessary public exposure  

---

## 🌱 Project Culture

We aim to build:

✔ responsible technology  
✔ healthy collaboration  
✔ real-world impact  

---

## 💬 Final Note

This is not just a code project.

It is a project that can impact real lives.

Act accordingly.
